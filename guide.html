အသုံးပြုပုံ
Gamepay Hub အသုံးပြုနည်းကို အဆင့်လိုက်ကြည့်နိုင်ပါတယ်

၁။ Game ရွေးပါ

ပထမဦးဆုံး main page ထဲက game card ကိုနှိပ်ပြီး မိမိ topup လုပ်မယ့် game ကိုရွေးပါ။

၂။ Item ရွေးပါ

လိုချင်တဲ့ item ကို + / - နဲ့ quantity ရွေးပါ။ Best Seller နဲ့ Unavailable badge တွေကိုလည်း ကြည့်နိုင်ပါတယ်။

၃။ Account ဖြည့်ပါ

Game ID / Server / Name ကို မှန်မှန်ကန်ကန် ဖြည့်ပေးပါ။ မှားရိုက်မိရင် topup မှားနိုင်လို့ သေချာစစ်ပေးပါ။

၄။ Payment ရွေးပါ

KPay / Wave / UAB / AyaPay စတဲ့ payment method ကိုရွေးပါ။ ရွေးပြီးရင် payment phone number နဲ့ name ပေါ်လာပါမယ်။

၅။ Order via Telegram နှိပ်ပါ

Order via Telegram ကိုနှိပ်ပြီး order စာကို Telegram မှာပို့ပါ။ ငွေလွှဲပြီးရင် payment screenshot ကိုပါ ပို့ပေးပါ။
